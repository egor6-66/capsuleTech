# 🚨 EWC

> **EWC** — рабочее пространство для списка инцидентов: общий список карточек, карта с маркерами и детальная карточка-форма. Один экран показывает данные сразу в трёх синхронизированных виджетах.

> **Аудитория этой страницы:** 👤 Юзер · 🛠️ Разработчик · 📊 Менеджер
> **Статус приложения:** 🟡 Demo / pre-prod _(данные — моки, бэкенд не подключён — см. [Дорожная карта](docs/03-roadmap/README.md))_

> 🖼️ **Скриншот:** _Дашборд — таблица инцидентов, карта, сайдбар-превью._ `docs/assets/dashboard-overview.png`

## 🚀 Запуск локально

Запускается из своей директории:

```bash
cd apps/ewc
pnpm dev
```

Демо-вход: **`user`** / **`123`** (см. [Авторизация](docs/01-features/auth.md)).

> [!NOTE]
> Данные — детерминированные моки (200 карточек, координаты в пределах одного города). Реальный API подключается в [Phase 2](docs/03-roadmap/README.md).
