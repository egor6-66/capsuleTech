# 📄 Открыть детальную карточку

> 🏠 [Хаб документации](../README.md) › 🧭 [How-to](README.md) › **Открыть детальную карточку**

> **Аудитория:** 👤 Пользователь
> **Статус:** ✅ Готово

Детальная карточка показывает всю информацию по происшествию — 5 секций полей. Открыть её можно тремя способами.

---

## Способы открыть

| Откуда | Действие |
|---|---|
| 📋 Таблица | **Двойной клик** по строке |
| 🗺️ Карта | **Двойной клик** по маркеру |
| 📄 Сайдбар | Выбери инцидент (одиночный клик) → нажми **«Открыть карточку»** |

✅ **Что увидишь:** откроется экран `/workspace/cards/:id` с детальной карточкой происшествия.

> [!NOTE]
> **Одиночный** клик только выбирает инцидент (подсветка + превью в сайдбаре). **Двойной** — сразу открывает карточку.

## Вернуться

Используй навигацию вверху (**Cards** / **Dashboard**) или кнопку «назад» браузера.

> [!TIP]
> Карточка пока **только для просмотра** — изменения не сохраняются. Почему так и что планируется — [Карточка происшествия](../01-features/incident-card.md#заглушки).
