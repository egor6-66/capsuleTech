## С чего обычно начинают

```
🔓 Включить режим редактирования
        │
        ├─→ ✋ переставить виджеты
        └─→ ↔️ изменить размеры

⚙️ Включить «умные» реакции (Widget settings)
        │
        ├─→ 🗺️ карта отлетает к выбранному
        └─→ 📋 таблица скроллится к выбранному
```

> 💡 Базовый выбор инцидента (одиночный клик подсвечивает, двойной — открывает карточку) работает **всегда**, без всякой настройки.
