# 🔓 Включить режим редактирования

> 🏠 [Хаб документации](../README.md) › 🧭 [How-to](README.md) › **Включить режим редактирования**

> **Аудитория:** 👤 Пользователь
> **Статус:** ✅ Готово

Режим редактирования делает виджеты «подвижными»: только в нём можно [переставлять](reorder-widgets.md) панели и [менять их размеры](resize-panels.md). В обычном режиме раскладка зафиксирована.

---

## Шаги

1. Открой меню в правом верхнем углу — иконка **☰**.
2. Найди группу **Layout**.
3. Нажми переключатель режима раскладки.

✅ **Что увидишь:** на виджетах появятся «ручки» перетаскивания и тянущиеся границы.

## Выключить

Тот же путь: **☰ → Layout → переключатель**. Раскладка зафиксируется в текущем виде.

> [!NOTE]
> Верхняя шапка (навигация, меню) намеренно **не** редактируется — двигать и ресайзить можно только рабочие виджеты дашборда.
