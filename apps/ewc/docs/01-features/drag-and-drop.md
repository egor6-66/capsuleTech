Раскладку можно настроить под себя: в режиме редактирования виджеты можно **менять местами** перетаскиванием. Таблица, карта и сайдбар входят в одну группу обмена.

## Что это и зачем

| 👤 Вы сделали | | ⚡ Что происходит |
|---|:---:|---|
| Включили режим редактирования (☰ → _Layout_) | → | 🔓 виджеты становятся «подвижными» |
| Перетащили один виджет на место другого | → | 🔄 виджеты меняются местами |
| Выключили режим | → | 🔒 раскладка зафиксирована |

Зачем: у разных людей разные привычки — кому-то карта важнее списка. Раскладка под себя ускоряет работу.

## Как работает под капотом

🛠️ _Для разработчиков._

### Слоты Matrix с `draggable` + `swapGroup`

Перетаскивание задаётся **декларативно** в конфиге слотов Matrix ([`pages/workspace/dashboard/index.tsx`](../../src/pages/workspace/dashboard/index.tsx)). Виджеты в одной `swapGroup` могут меняться местами:

```ts
slots={{
  main:    { children: <Widgets.Tables.Incidents />, draggable: true, swapGroup: 'widgets' },
  rightBar:{ children: <Widgets.Sidebars.Main />,     draggable: true, swapGroup: 'widgets' },
  footer:  { children: <Widgets.Maps.World />,        draggable: true, swapGroup: 'widgets' },
}}
```

Все три — в группе `'widgets'`, поэтому таблица ↔ карта ↔ сайдбар взаимозаменяемы.

### Режим редактирования — глобальный тумблер

DnD активен только в edit-режиме. Тумблер — `Ui.LayoutModeToggle` в [`views/workspaceMenu.tsx`](../../src/views/workspaceMenu.tsx) (группа _Layout_). Он переключает глобальный `layoutMode` в сторе `@capsuletech/web-style`; Matrix сам его подхватывает.

> [!IMPORTANT]
> Workspace-shell ([`pages/workspace/index.tsx`](../../src/pages/workspace/index.tsx)) намеренно стоит на `layoutMode="view"` — каркас (header/main) **не** должен ресайзиться или таскаться. Подвижен только внутренний Matrix дашборда.

### Движок

Под капотом — `@capsuletech/web-dnd` (pointer-based DnD для Solid: mouse + touch без HTML5-флэйков). EWC его напрямую не трогает — всё идёт через слот-конфиг Matrix.

## How-to — переставить виджеты

👤 _Пошагово._

1. Открой меню (☰) → группа **Layout** → включи режим раскладки.
2. Наведись на виджет, зажми и перетащи на место другого.
3. Отпусти — виджеты поменяются местами.
4. Выключи режим, чтобы зафиксировать.

📎 Детальный разбор: [How-to → Переставить виджеты](../02-how-to/reorder-widgets.md).
