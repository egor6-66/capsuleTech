У виджетов есть **полоска настроек** (settings-strip), которая появляется по глобальному тумблеру. В ней — переключатели «умных» реакций синхронизации: подлетать ли карте, скроллиться ли таблице.

## Что это и зачем

Два уровня:

| Уровень | Где | Что делает |
|---|---|---|
| **Глобальный тумблер** | Меню (☰) → _Widget settings_ | Показывает/прячет полоски настроек на всех виджетах |
| **Настройка виджета** | Полоска на карте / таблице | Включает конкретную реакцию синхронизации |

Доступные настройки:

| Тумблер | На виджете | Что включает |
|---|---|---|
| 🗺️ **Подлетать к выбранному** | Карта | Камера отлетает к маркеру при выборе строки в таблице |
| 📋 **Скроллить к выбранному** | Таблица | Таблица прокручивается к строке при клике по маркеру |

> [!NOTE]
> Тумблеры независимы и по умолчанию **выключены** — «умные» реакции опциональны, чтобы не мешать, когда не нужны.

## Как работает под капотом

🛠️ _Для разработчиков._

### Тумблер — это тоже тег

Кнопки настроек ничего не мутируют сами. Они несут meta-тег, а флипает флаг всё тот же `onClick`-роутер фичи:

| Контрол | Тег | Флипает |
|---|---|---|
| [`views/settings/mapSync.tsx`](../../src/views/settings/mapSync.tsx) | `toggle-fly` | `flyToSelected` |
| [`views/settings/tableSync.tsx`](../../src/views/settings/tableSync.tsx) | `toggle-scroll` | `scrollToSelected` |

```ts
// views/settings/mapSync.tsx — View читает флаг, не мутирует
const on = () => !!ctx.store.ctx.data?.flyToSelected;
<Ui.Button variant={on() ? 'default' : 'outline'} meta={{ tags: ['toggle-fly'] }}>
  {on() ? '✓ Подлетать к выбранному' : 'Подлетать к выбранному'}
</Ui.Button>
```

```ts
// features/incidents.ts:108 — роутер флипает
if (tags.includes('toggle-fly')) {
  store.update({ flyToSelected: !store.ctx.data.flyToSelected });
}
```

### Где монтируются

Полоски настроек подаются в `settings`-слоты Matrix на дашборде ([`pages/workspace/dashboard/index.tsx`](../../src/pages/workspace/dashboard/index.tsx)):

```ts
main:   { children: <Widgets.Tables.Incidents />, settings: <Views.Settings.TableSync />, ... }
footer: { children: <Widgets.Maps.World />,       settings: <Views.Settings.MapSync />,  ... }
```

Глобальный тумблер — `Ui.WidgetSettingsToggle` в [`views/workspaceMenu.tsx`](../../src/views/workspaceMenu.tsx) (группа _Widget settings_). Он управляет видимостью settings-слотов через стор `@capsuletech/web-style`.

## How-to — включить настройки

👤 _Пошагово для пользователя._

1. Открой меню в правом верхнем углу (☰).
2. В группе **Widget settings** включи тумблер → на виджетах появятся полоски настроек.
3. На нужном виджете нажми тумблер реакции (станет `✓`).
