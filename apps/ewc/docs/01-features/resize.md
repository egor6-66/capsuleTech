## Что это и зачем

| 👤 Вы сделали | | ⚡ Что происходит |
|---|:---:|---|
| Подвели курсор к границе панелей | → | ↔️ курсор меняется на «тянущий» |
| Потянули границу | → | 📐 соседние панели меняют размер |

Стартовые размеры заданы заранее, дальше — под себя. Например, сайдбар стартует с ¼ ширины, карта — с ⅓ высоты.

## Как работает под капотом

🛠️ _Для разработчиков._

### `initialSize` + `resizable` в слотах Matrix

Ресайз — декларативное свойство слота. Стартовая доля задаётся `initialSize` (0..1), запрет ресайза — `resizable: false`:

```ts
// pages/workspace/dashboard/index.tsx
rightBar: { children: <Widgets.Sidebars.Main />, initialSize: 0.25, ... }  // ¼ ширины
footer:   { children: <Widgets.Maps.World />,    initialSize: 0.35, ... }  // ⅓ высоты
```

```ts
// pages/workspace/index.tsx — каркас не ресайзится
header: { children: <Widgets.Headers.Main />, resizable: false, initialSize: 0.04 }
main:   { children: <Ui.Outlet />,            resizable: false }
```

### Кто несёт ресайз

Ресайз — встроенная способность `Ui.Layout.Matrix` (UI-kit). Как и [DnD](drag-and-drop.md), он активен в edit-режиме (`layoutMode`), который переключается тумблером _Layout_ в [меню](../../src/views/workspaceMenu.tsx). EWC задаёт только `initialSize`/`resizable` — остальное внутри Matrix.

## How-to — изменить размеры

👤 _Пошагово._

1. Открой меню (☰) → группа **Layout** → включи режим раскладки.
2. Наведись на границу между панелями (курсор станет «тянущим»).
3. Зажми и потяни — соседние панели изменят размер.

📎 Детальный разбор: [How-to → Изменить размеры панелей](../02-how-to/resize-panels.md).
