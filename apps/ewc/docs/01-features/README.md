## Как читать страницу фичи

Каждая страница построена одинаково:

| Секция | Для кого |
|---|---|
| **Что это и зачем** | 📊 Менеджер · 👤 Юзер — простыми словами, действие → результат |
| **Как работает под капотом** | 🛠️ Разработчик — со ссылками на код `file:line` |
| **How-to** | 👤 Юзер — пошаговый рецепт (где применимо) |
| **Шероховатости** | 🛠️ · 📊 — ограничения и известные баги |
