## Что это и зачем

| 👤 Вы сделали | | ⚡ Что происходит |
|---|:---:|---|
| Смотрите на карту | → | 📍 видны маркеры всех происшествий на карте города |
| **Клик** по маркеру | → | ✅ инцидент выбран — маркер активен, таблица и сайдбар реагируют |
| **Двойной клик** по маркеру | → | 📄 открывается детальная карточка |
| Выбрали инцидент в таблице _(если включён тумблер)_ | → | 🎬 камера плавно отлетает к маркеру |

Карта живёт в двух местах: как footer-виджет дашборда и как отдельный экран `/workspace/map` на весь экран.

## Как работает под капотом

🛠️ _Для разработчиков._

### Виджет + Stateless-маркеры

- **Widget** [`widgets/maps/world.tsx`](../../src/widgets/maps/world.tsx) — настраивает `Ui.MapView` (центр `[30.3158, 59.9311]`, zoom `13`), читает `items` из стора и считает `flyTo`.
- **View** [`views/markersList.tsx`](../../src/views/markersList.tsx) — stateless, итерирует `items` через Solid `<For>` и рисует `Ui.MapView.Marker`. Каждый маркер несёт `meta.tags = ['incident', 'map']` + `payload.id`.

```ts
// views/markersList.tsx
<For each={props.items ?? []}>
  {(incident) => (
    <Ui.MapView.Marker
      lng={incident.location.lng}
      lat={incident.location.lat}
      active={incident.id === props.activeId}
      meta={{ tags: ['incident', 'map'] }}
      payload={{ id: incident.id }}
    />
  )}
</For>
```

### «Подлёт» камеры (flyTo)

`flyTo` — это анимация (в отличие от `center`, который прыгает). Виджет отдаёт координаты в `<Ui.MapView flyTo={...}>` **только** когда выбор пришёл из другого виджета:

```ts
// widgets/maps/world.tsx:23
data()?.flyToSelected && sel && data()?.selectionSource !== 'map'
  ? [sel.location.lng, sel.location.lat]   // ← летим к маркеру
  : undefined                              //   свой клик — не двигаемся
```

Гейт по тумблеру `flyToSelected` — см. [⚙️ Настройки виджетов](widget-settings.md) и [🔗 Синхронизация](map-table-sync.md).

### Лоадер без мигания тайлов

Под лоадером (`store.loading`) Widget рисует map-скелетон. `<Show>` в WidgetWrapper **не монтирует** контент-ветку под лоадером — поэтому тяжёлый MapLibre-инстанс не создаётся раньше времени:

```ts
(Ui) => <Ui.Skeleton variant="map" />
```
