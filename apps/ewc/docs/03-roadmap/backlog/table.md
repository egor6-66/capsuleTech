# 📋 Бэклог — Таблица

> 🏠 [Хаб документации](../../README.md) › 🗺️ [Дорожная карта](../README.md) › 💡 [Бэклог](README.md) › **Таблица**

> **Аудитория:** 🛠️ Разработчик · 📊 Менеджер
> **Статус:** 📋 Идеи с оценками

Что можно добавить в [таблицу инцидентов](../../01-features/table.md). Сейчас это `Ui.DataTable` с сортировкой и `plain`-пагинацией. Большинство пунктов — расширение конфига Shape, часть — фреймворковая работа в `@capsuletech/web-ui`.

---

## Идеи

| 💡 Фича | Оценка | Что даёт | Зависимости |
|---|:---:|---|---|
| **Ресайз колонок** | 🟡 M (2–3) | Тянуть границу заголовка колонки | `web-ui` DataTable column-resize |
| **Добавить/убрать/переставить колонки** | 🟡 M (3–4) | Настройка видимых полей под задачу | column-visibility API + persist |
| **Фильтр / поиск** | 🟡 M (2–3) | Строка поиска + фильтры по колонкам | предикат над `store.ctx.items` |
| **Группировка** | 🟡 M (3–4) | Свернуть по району / типу / дате | group-by в DataTable |
| **Экспорт (CSV / Excel)** | 🟢 S (1–2) | Выгрузка текущей выборки | сериализатор + download |
| **Сохранение настроек таблицы** | 🟢 S (1–2) | Колонки/сортировка переживают перезагрузку | `localStorage` / layout-persist |
| **Починка виртуального скролла** | 🏗️ | Большие списки без `plain`-режима | ⛔ [баг фреймворка](../known-issues.md#виртуальный-скролл-таблицы) |

**Сумма app-части (без 🏗️):** ~12–18 дней.

---

## Заметки по реализации

> [!WARNING]
> **Виртуальный скролл — НЕ app-задача.** Пункт помечен 🏗️: его чинит `owner-web-ui` в [`@capsuletech/web-ui`](../tech-debt.md). Пока не починен — таблица на `plain`-режиме ([почему](../known-issues.md#виртуальный-скролл-таблицы)). На моках (200 строк) разницы нет; виртуализация понадобится только на тысячах реальных строк — к тому моменту фикс уже будет.

> [!TIP]
> **Фильтр таблицы = фильтр карты.** Один предикат над `store.ctx.items`, два потребителя. Планировать как [общую фичу с картой](map.md).

> [!NOTE]
> **Большинство пунктов — конфиг Shape, а не новый код.** Колонки, сортировка, группировка живут в [`shapes/incidentsTable.tsx`](../../00-overview/architecture.md). Расширяется декларативно — отсюда низкие оценки. «Дорогие» места — там, где нужен новый примитив в `web-ui`.
