# 📊 Для менеджеров

> 🏠 [Хаб документации](../README.md) › **Для менеджеров**

> **Аудитория:** 📊 Менеджер · 🛠️ Тимлид
> **Статус:** 🟡 EWC на этапе Demo / pre-prod

Раздел без кода. Отвечает на три управленческих вопроса: **что можем показать прямо сейчас**, **что мешает двигаться дальше** и **сколько примерно стоит каждый следующий шаг**.

---

## Краткая сводка для занятых

| Вопрос | Ответ в одну строку |
|---|---|
| 🎬 **Можно показать?** | Да — рабочий экран с тремя синхронизированными виджетами на демо-данных. |
| 🟢 **Что точно работает?** | Синхронизация виджетов, таблица, карта, перетаскивание/ресайз панелей, вход. |
| 🟡 **Что с оговорками?** | Карточка происшествия (только просмотр), авторизация (демо-логин). |
| 🔴 **Чего ещё нет?** | Реальных данных, сохранения изменений, защиты входа. Это [Phase 2](../03-roadmap/README.md#этапы). |
| ⏱️ **До «боевого» состояния?** | Порядок — недели, не дни. Зависит от готовности бэкенда. Детали — [матрица стоимости](cost-matrix.md). |

> [!IMPORTANT]
> **Часть работы — не на стороне команды приложения.** EWC построен на базовой платформе. Некоторые улучшения (например, ускорение больших таблиц) делает команда платформы — для бизнеса это значит, что они «придут сами» с обновлением платформы, без отдельного проекта. Такие пункты помечены 🏗️.

## Чего ждать на демо

| Сценарий | Готовность | Комментарий |
|---|:---:|---|
| Обзор происшествий на карте и в таблице | ✅ | 200 демо-карточек |
| «Кликнул здесь — подсветилось везде» | ✅ | Ключевой приём, надёжно работает |
| Перекроить раскладку под себя | ✅ | Drag-and-drop + ресайз |
| Открыть детальную карточку | 🟡 | Открывается, но только просмотр |
| Вход в систему | 🟡 | Демо-логин `user` / `123` |

> [!TIP]
> Полный разбор каждого пункта с бизнес-ценностью — [Возможности «сейчас»](capabilities.md).
