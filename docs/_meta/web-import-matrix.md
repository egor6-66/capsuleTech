**Legend (zone canon, ADR 047):**
- `kit` — stateless primitives (ui)
- `runtime` — framework runtime (core, state, router, …)
- `domain` — domain-modules (auth, shell, agent)
- `boost` — domain-boosters (table, map, flow, chart, layout)
- `studio` — design-time host (sole inhabitant)

Direction discipline (ADR 047):
- `runtime ← kit`: OK (runtime may use kit primitives)
- `domain ← runtime, kit`: OK
- `boost ← runtime, kit`: OK
- `studio ← *`: OK (top consumer)
- `kit → *`: violation candidate (kit must stay leaf)
- `runtime → domain | boost | studio`: violation candidate
- `domain → boost | studio` or `boost → domain | studio`: cross-axis violation candidate
