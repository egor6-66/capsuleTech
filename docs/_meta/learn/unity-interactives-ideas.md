# 3D-интерактивы (Unity) — идеи под наш движок + нейронку

> [!abstract] Принцип: **интерактив = МЕХАНИКА поверх существующего контента**
> (senses/drills/`nearMiss`/`graboTag`/уровни), а не отдельный контент. 3D берём
> только там, где плоские карточки бессильны. Нейронка = генерация + оценка +
> голос + ассеты. Связано с [[brief-app-architecture]], [[brief-learning-concepts]].

## Что 3D/нейронка дают, чего нет на карточках
- **Пространство** — предлоги места, движение, направление (on/under/into) физически.
- **Время как ось** — наша модель «время × вид» становится играбельной шкалой.
- **Голос** — говорение и связная речь (STT оценивает флэп/редукции), не только текст.
- **Живой собеседник** — NPC на LLM: свободная продукция под контролем цели урока.
- **Бесконечная вариативность** — генерация новых заданий из одного `graboTag`.

## Идеи (каждая привязана к нашему принципу)

### ⭐ 1. Sandbox предлогов места (3D-native)
- **Учит:** on / under / in / behind / next to / into — то, что на плоскости мертво.
- **Механика:** сцена с коробкой/полкой/столом; задание «Put the cat **on** the box»
  (текст или голосом) → тащишь 3D-объект → игра проверяет позицию.
- **Нейро:** STT (можешь СКАЗАТЬ фразу), LLM-судья на свободный ответ; image/asset-gen
  для объектов из поля `image` смысла.
- **Данные:** правило `grammar-prepositions`, слова с `image`.

### ⭐ 2. Ось времён — играбельная шкала (наша модель во плоти)
- **Учит:** выбор времени = «где якорь + нужен ли дотяг» ([[brief-learning-concepts]]).
- **Механика:** рельс past — now — future; ставишь пузырь-действие на точку +
  переключатель вида (simple/continuous/perfect). Сцена **проигрывает** ситуацию
  (человек делал вчера / делает сейчас / уже сделал). Собрал позицию → собери фразу.
- **Нейро:** LLM генерит целевую фразу и валидирует; сцена рендерит ситуацию.
- **Данные:** `grammar-verbs-tenses`, дриллы `past-continuous-vs-simple` и т.п.

### ⭐ 3. NPC-ролплей (нейро-собеседник, говорение)
- **Учит:** свободная продукция (наш канон «production, не выбор»).
- **Механика:** 3D-персонаж, сценарий (заказать кофе, спросить дорогу). Голос:
  STT → LLM (в роли) → TTS. LLM держит характер И оценивает против цели урока
  (напр. «нужно использовать Past Continuous»), подсвечивает `nearMiss`-грабли.
- **Нейро:** ядро — STT+LLM+TTS. Тут нейронка обязательна.
- **Данные:** урок задаёт цель (rule/graboTag), словарь — лексику сцены.

### 4. Сцена песни / караоке (наш якорь — песни)
- **Учит:** связная речь (флэп, редукции, g-dropping), лексика из треков.
- **Механика:** 3D-сцена, строки летят по биту; поёшь/проговариваешь, STT скорит
  произношение; пропущенное слово надо вставить. (Егор делает эмо-скримо-группу —
  заходит идеально 🖤)
- **Нейро:** STT-оценка произношения; выравнивание по фонемам (wav2vec2 из
  [[brief-voice-module]]).
- **Данные:** `connected-speech`, песенный скоуп слов.

### 5. Grabla-hunter (грабли как враги)
- **Учит:** ловит L1-интерференцию; геймифицирует каталог `graboTag` + spaced-возврат.
- **Механика:** ошибка-калька летит на тебя монстром («we with him») → бьёшь верной
  формой («He and I»). Личные грабли юзера возвращаются боссами (spaced, слой аккаунта).
- **Нейро:** генерит свежие предложения на данный `graboTag` (бесконечно).
- **Данные:** прямо наши дриллы + `nearMiss` (готовые «неверные» = снаряды врага).

### 6. Опиши сцену (image → продукция)
- **Учит:** сборка фразы, лексика; обратка к «слово = образ».
- **Механика:** нейронка генерит 3D-сцену → опиши по-английски (голосом/текстом) →
  LLM грейдит и подсказывает.
- **Нейро:** сцена-ген + LLM-судья + STT.

### 7. Мир образов — «сыграй глагол» (word-as-image в 3D)
- **Учит:** глаголы действия и их формы через ДЕЙСТВИЕ, не перевод.
- **Механика:** pour / open / fall / push — выполняешь действие над 3D-объектом,
  подкрепляется глагол + формы (past/participle). Концепт [[word-as-image]] буквально.

## Архитектурная склейка (важное для юнити-архитектора)

1. **Контент не дублируем.** Unity — ещё один *потребитель* той же базы (senses,
   drills, слова с `image`/`pron`/audio). Игра = рендер+механика над данными дрилла.
   Драйвится тем же `promptRu/answerEn/accept/nearMiss/graboTag/words`.
2. **Роли нейронки:** (а) генерация заданий/сцен из `graboTag`/набора слов;
   (б) оценка свободной продукции; (в) голос TTS↔STT; (г) 3D-ассеты из текста-образа.
3. **Оценка — тирами (дёшево→дорого):** сначала детерминированно (`contains`/`regex`
   из `nearMiss` — мгновенно, бесплатно), LLM-судья только для открытых ответов.
   Переиспользуем наш `nearMiss`-канон как быстрый путь.
4. **Сложность/маршрут:** берём `level` (L0–L5) и `graboTag`; spaced-возврат — слой
   аккаунта (личные грабли), как договорено, контент про это не знает.
5. **Голос:** стыкуется с [[brief-voice-module]] (wav2vec2-phoneme + espeak-ng) —
   для скоринга произношения в песнях/предлогах.

## С чего начать (макс. уникальная ценность, чего плоское НЕ может)
1. **Sandbox предлогов** — 3D-native, мгновенный «вау», малый скоуп.
2. **Ось времён** — превращает нашу главную модель в игру.
3. **NPC-ролплей** — единственный путь к реальному говорению.
Остальное (песни, grabla-hunter) — второй эшелон, как дозреют голос и каталог граблей.
