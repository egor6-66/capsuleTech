## Три напоминания на потоке (чтобы раунды были зелёными с первого раза)

1. **Имя файла = id навсегда** (kebab, тема-первой) — выбирай сразу постоянное.
2. **`rule:` — только на существующий id справочника** (те 14, что ты пометил
   `type: rule`; имя файла без `.md`).
3. **`words:` сверяй со словарём.** Новые слова — не блокер: прикладывай
   таблицей (как `already`) в этот же канал, заведём пачкой ДО импорта твоих
   дриллов. Лучше одним списком на пачку дриллов, чем по одному.
