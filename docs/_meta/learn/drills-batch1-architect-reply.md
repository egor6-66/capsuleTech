# Пачка 1: ответы на оба нюанса + что нужно для зелёного импорта

## Ответы на твои вопросы

1. **Регистр — да, как ты рассчитывал.** Проверка нормализует ввод и эталоны:
   trim → апострофы → **lowercase** → пробелы → финальная пунктуация. Паттерны
   в нижнем регистре — канон, ничего не добавляй.
2. **Переформулировка валидатора принята**: «промпт однозначен для
   ПРОВЕРЯЕМОГО ИЗМЕРЕНИЯ» — это теперь текст правила №2 (для времён —
   маркер/контекст; для местоимений/артиклей достаточно недвусмысленности).
   Код пока не трогаем — твой приём с `context` на не-tense item'ах штатный;
   ослабим валидатор, когда короткие императивы начнут реально мешать.
3. **Функциональные слова НЕ в `words[]`** — правильно, так и оставляем.

## Почему в аппе раздел Lessons пустой

Конвейер живой, но списку нечего отдавать: **в vault нет ни одного файла
`type: lesson`** (и concept-файлов тоже — `word-as-image` пока не лежит в
`lessons/concepts/`). Урок = сшивка, без неё дриллы/правила в разделе не
появляются. Просьба к пачке 1 добавить:
1. `lessons/concepts/word-as-image.md` (твой seed-концепт, поля — финалочка);
2. хотя бы один `lessons/*.md` — например `pronouns-basics`: concept →
   `grammar-pronouns` → дрилл `subject-object-pronouns`. Любой на твой вкус —
   важно, чтобы в аппе появился первый живой путь.
